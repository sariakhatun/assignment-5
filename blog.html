Question-1: What are the different ways to select an element in the DOM?

Answer : There are 6 ways to select an element in the DOM. Such as,
1.getElementById() => "select element by id"
2.getElementsByClassName() => "select element by class"
3.getElementByTagName() => "select element by tag name"
4.querySelector() => "select the first element using css selector"
5.querySelectorAll() => "select the all matching element using css selector"
3.getElementByName() => "select element by name attribute , usually used in input field"

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer :
1.innerHTML : gets or sets the html content
2.innerText : gets or sets only visible text
3.textContent : gets or sets all text, including hidden content

Question-3: What is event delegation in the DOM?

Answer : Event Delegation is a technique where a parent element handles events for its child elements, even if they are added dynamically

Question-4: What is event bubbling in the DOM?

Answer : Event Bubbling is a mechanism where an event starts form the target element and moves up to its ancestor(parent,gradparent, etc) in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer :
1.Create an element : document.createElement("tagname")
2.Add an element : document.appendChild(element)
3.Remove an element : document.remove()
